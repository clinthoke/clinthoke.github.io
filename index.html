Capstone: Predicting Disney World wait times

Python, Pandas, Numpy, FBprophet, Linear Regression

This project was my capstone for General Assembly Data Science Immersive. The capstone involved using multiple regression models as well as FBprophet to predict the wait times for rides at the Walt Disney World theme parks in Orlando Florida. The majority of the code is broken up into three parts:

  • Identifying the time period and performing EDA and data cleaning.
  • Modeling 14 rides with 8 regression methods to find the lowest wait times.
  • Using only time data, made predictions with the FBprophet model.
All the code was written in python. The data for the wait times was provided by touringplans.com.
